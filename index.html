The Game is Broken/Isn't Working

If the game isn't working on this website, make sure to click the "Report a Problem" button thats on the bottom of the screen of most pages and fill out the short survey. Make sure to include detail about which game is having issues and exactly what the problem is.

Can You Add This Update?

If you want me to add an update to the game, click the "Request" button which is in FAQ on the nnavbar and fill out the short survey. Please remember that requesting an update more than once won't make it get added faster.
